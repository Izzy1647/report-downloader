Getting Started

The Adyen Report Downloader Web App allows you to bulk-download reports from multiple merchant accounts efficiently. Here's what you need to get started:

Prerequisites

  • Adyen Report User API Key: You need an API key with Report User permissions
  • Account Structure JSON File: A file organizing your merchant accounts into groups
  • Modern Web Browser: Chrome, Firefox, Safari, or Edge
Tip: You don't need to install anything! This is a web-based application that runs entirely in your browser.

JSON File Structure

Your account structure file organizes merchant accounts into logical groups. This makes it easy to download reports for specific business units, regions, or account types.

Quick Start: You can download a sample JSON file below and modify it with your actual merchant accounts.

This example shows how to organize merchant accounts by region and channel:

{ "North America - Retail Stores": [ "COMPANY-US-NewYorkFifthAve", "COMPANY-US-LABeverlyCenter", "COMPANY-CA-TorontoPremium" ], "North America - E-commerce": [ "COMPANY-US-ECOM", "COMPANY-CA-ECOM", "COMPANY-US-AMAZON" ], "Europe - Physical Stores": [ "COMPANY-UK-LondonOxfordSt", "COMPANY-FR-ParisChamps", "COMPANY-DE-BerlinKurfurstendamm" ], "Asia Pacific - Retail": [ "COMPANY-SG-OrchardRoad", "COMPANY-HK-CentralIFC", "COMPANY-JP-TokyoGinza" ] }
Pro Tip: The downloadable sample file contains a more comprehensive example with 8 regional groups and 40+ merchant accounts across different channels and geographies.

Important Rules

  • Group Names: Use descriptive names that help you organize accounts (e.g., regions, business units)
  • Merchant Accounts: Use exact Adyen merchant account codes
  • Format: Must be valid JSON with proper quotes and commas
  • Empty Groups: Groups can have one or more merchant accounts
Important: Merchant account names must match exactly with your Adyen account configuration. Check your Adyen Customer Area for the correct merchant account codes.

Step-by-Step Guide

  1. Prepare Your API Key

    Get your Adyen Report User API key from the Customer Area. This key should have permissions to access reports for all merchant accounts you want to download from.

  2. Create Account Structure File

    Create a JSON file organizing your merchant accounts into groups. Use the structure shown in the JSON Structure section above.

  3. Open the Web App

    Navigate to the Adyen Report Downloader web app in your browser.

  4. Enter Your API Key

    In the Configuration section, enter your Adyen Report User API key. The key will be hidden for security.

  5. Import Account Structure

    Click the "Import JSON" button and select your account structure file. The app will validate the file and display your groups.

  6. Select Target Month

    Choose the month for which you want to download reports. The app will filter reports to only include files from this month.

  7. Choose Report Types

    Select which types of reports you want to download. Available types include Monthly Finance, Invoice Overview, Daily Payment Accounting, and Settlement Detail.

  8. Select Groups

    Check the boxes next to the groups you want to include in this download. You can use "Select All" or "Deselect All" for convenience.

  9. Start Download

    Click the "Start Download" button. The app will show real-time progress as it downloads reports.

  10. Download ZIP File

    Once the download is complete, click "Download ZIP File" to get all reports in a single ZIP file with organized folder structure.

API Key Setup

Getting Your API Key

  1. Log in to your Adyen Customer Area
  2. Navigate to Developers > API credentials
  3. Create a new API key or use an existing one
  4. Ensure the key has "Report User" role
  5. Copy the API key and keep it secure

Required Permissions

Your API key needs the following permissions:

  • Report User Role: Access to reports endpoint
  • Merchant Account Access: Permissions for all merchant accounts in your structure
  • Read Permissions: Ability to download report files
Security Note: Your API key is processed server-side and never stored. It's only used during the download session and then discarded.

Report Types

The app supports four types of Adyen reports. Each report type is stored in its own folder within the ZIP file.

Monthly Finance

Monthly financial summaries including revenue, fees, and settlements for each merchant account.

Invoice Overview

Invoice documents showing billing information and charges for the period.

Daily Payment Accounting

Daily transaction details including payments, refunds, and chargebacks.

Settlement Detail

Detailed settlement information showing how funds are distributed to bank accounts.

Folder Structure in ZIP

reports_YYYYMM/ Group Name/ Merchant Account/ monthly_finance/ invoice_overview/ daily_payment_accounting/ settlement_detail/

Troubleshooting

Common Issues

Download Fails with "No Files Found"

  • Check if the selected month has available reports
  • Verify merchant account codes are correct
  • Ensure API key has proper permissions
  • Try different report types

API Key Authentication Error

  • Verify API key is correct and active
  • Check if key has Report User role
  • Ensure key has access to selected merchant accounts

JSON Import Fails

  • Validate JSON syntax using online validator
  • Ensure proper quotes around all strings
  • Check for trailing commas
  • Verify file is saved with .json extension

Progress Stuck or Slow

  • Large downloads may take time
  • Check network connection stability
  • Try downloading smaller groups first
  • Monitor browser console for errors

Getting Help

If you continue to experience issues:

  • Check the browser console for error messages
  • Verify all input parameters are correct
  • Test with a small group first
  • Contact your Adyen support team for API-related issues
